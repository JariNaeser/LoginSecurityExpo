Web Security Expo



Usage



This application does not have many functionalities as it's intended to display how such a secure system works. Its usage is simple, a registered user can view the data relatative to himself including his salary. Only an admin can see the data of every registered user.

Go to login


About


The scope of this little project is to secure the communication between a client and a server. The technologies used to achieve this are the following:

  • Angular (frontend)
  • Java Spring MVC (backend)
  • MySQL (Database)
  • JWT (authentication)
  • HTTPS + Certificates (communication encryption)
